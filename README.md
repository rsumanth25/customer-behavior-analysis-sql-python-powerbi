# 📊 Customer Behavior Analysis using SQL, Python & Power BI

## 📌 Project Overview

This project focuses on analyzing customer purchasing behavior to generate actionable business insights. The analysis is performed using Python for data preprocessing, SQL for querying, and Power BI for visualization.

The goal is to understand customer trends, identify high-value segments, and support data-driven decision-making.

---

## 🎯 Business Objectives

* Identify high-value customers and their characteristics
* Analyze revenue contribution across different customer segments
* Evaluate the impact of discounts and subscription status on purchasing behavior
* Understand seasonal and category-wise sales trends
* Provide actionable recommendations for business growth

---

## 🛠️ Tools & Technologies

* **Python** (pandas, numpy) – Data cleaning & preprocessing
* **SQL (PostgreSQL)** – Data querying & analysis
* **Power BI** – Dashboard & visualization
* **Jupyter Notebook** – Development environment

## 🔄 Project Workflow

1. **Data Collection & Cleaning**

   * Loaded dataset using Python
   * Handled missing values and duplicates
   * Performed feature engineering (e.g., age groups, high-value customers)

2. **Exploratory Data Analysis (EDA)**

   * Analyzed customer demographics and purchasing patterns
   * Identified trends across categories, seasons, and regions

3. **SQL Analysis**

   * Wrote queries to answer key business questions:

     * Revenue by age group and category
     * Customer segmentation
     * Discount impact analysis
     * Subscription vs non-subscription behavior

4. **Dashboard Creation**

   * Built interactive Power BI dashboard
   * Included KPIs, filters, and trend visualizations

---

## 📊 Key Insights

* Customers in specific age groups contribute significantly to total revenue
* Subscription-based customers show higher average spending
* Discounts influence purchasing behavior but vary across categories
* Certain product categories dominate overall sales
* Seasonal trends impact revenue patterns
